HOW TO RUN

Reconstruction Analysis:
1) Go into the run folder, open root
2) type: .L sborn_analysis_3_6_2020.C
3) type: AnalyzeVertices(#) 
	- # = 310 or 3122 for kaons or lambdas 
4) Find out output root file in the outputs folder


Truth Plots:
1) Go into the run folder, open root
2) type: .L sborn_truth_plots_feb.C
3) type: AnalyzeVertices(#) 
	- # = 310 or 3122 for kaons or lambdas 
4) Find out output root file in the outputs folder

Armenteros Podolanski Cut Depictions:
1) Go into the run folder, open root
2) type: .L arm_cut_depcitons.C
3) type: run(#) 
	- # = 310 or 3122 for kaons or lambdas 
	- Never fully set this up, so it only works for kaons
4) Find out output root file in the outputs folder
